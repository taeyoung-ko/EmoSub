# cxd

### To run
1. python server.py
2. open front.html

### Explanation
- record.py (record voice)
- emotion.py (speech emotion recognition)
- transcribe.py (speech to text)
- server.py (combine tasks & open server)
- fromt.html (demo /w (LeonSans)[https://github.com/cmiscm/leonsans])
